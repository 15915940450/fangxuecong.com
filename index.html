some projects

  • 让机器自己去玩的俄罗斯方块

    AI俄罗斯方块，2014年的作品，目的是为了强化js编程。采用了Pierre Dellacherie算法，此算法只考虑当前方块，不对未来的情况进行计算，算法每次生成一个方块，便穷举该方块所有旋转的所有落点。通过一个评估函数，选择最优的下落位置。

  • 照片增强项目

    使用AI算法将像素化、模糊和低质量的照片转换为高质量清晰的照片。熟练的vue3, element plus开发技术，运用了canvas+fabric，TWEEN动画库等技术点，完成PC与移动端适配的复杂交互的应用，直观的展示了前后照片对比效果。其中有不少数理变量值的转换公式，通过画示意图，悉心专研，充分结合矩阵变换等数学知识推导，应用到实际开发。

  • 算法

    身为程序员，对算法多多少少得要了解一些。通读《算法（第4版）》之后，于业余时间用 JavaScript 写过不少经典算法问题。如 八皇后问题 ， 马踏棋盘 ， 迷宫生成算法与迷宫寻路算法 ， AI:俄罗斯方块(Tetris) ， 0-1背包 ， 数独 ， 快速排序 ， 旅行商问题 等。

  • 代理商管理后台

    易马达业务迅速扩展，为其代理商提供一个管理工具。最初1.0.0版本前端用了13个工作日的开发周期，其中运用到vuejs，elementUI技术。

  • 中控故障返修平台

    平台所实现的主要功能有：账号登录，可根据sid(mac)、SN查询中控，展示与该中控有关的故障信息，处理故障，登记报废。所采用的技术链为：React+Material-UI。

  • 俄罗斯方块

    没有尝试自己写过一个游戏，算不上热爱编程。还记得小时候我们一班农村孩子排排坐，拿着小游戏机就是为这俄罗斯方块着迷。

    该项目是一个业余项目，主要是在canvas上绘图，还涉及一些PHP，MySQL知识，用来保存游戏分数，统计pv之类。

庄子说过：“水之积也不厚，则其负大舟也无力。”作为一名前端开发工程师，我对前端的基础知识 HTML ， CSS ， JavaScript 驾轻就熟，清楚整个网站开发流程。

刚毕业的那一两年，我从事网页设计工作，用AI和Photoshop把需求展现在效果图之上。

从2013年开始，我便专注于前端领域，经过多年的工作与学习，让我在 jQuery 、 Bootstrap 、 Node.js 、 D3.js 、 Snap.svg 、 PHP 等技术点都有涉略。我偏向于 git 工具控制项目版本，我习惯于开源免费的 Atom 现代编辑器。此外我每天都用 firebug 来调试网页，RESTClient、ESlint、Fiddler这些好工具有时真帮了我大忙。下班回到家，我就在个人电脑玩转Ubuntu系统，对Linux常用命令、终端 VIM 不会感到陌生。

这几年前端发展很快，在最近的一年时间里，我着重学习和运用 Vue.js ， React 框架，结合 webpack 热开发，打包发佈，使用 gulp 构建自动化工作流。

我注重细节，拥有优雅的编程风格 ...

代码就是财富。代码是写给人看的，为方便其他同事阅读、维护，我在开发过程中，除了编写注释之外，通常牢记简单的两点：

  • 缩进 -- 两个空格
  • 命名 -- class使用减号（select-all-btn），js变量使用驼峰（eleSelectAllBtn）

... 以及敏捷的逻辑思维
