庄子说过：“水之积也不厚，则其负大舟也无力。”作为一名前端开发工程师，我对前端的基础知识 HTML ， CSS ， JavaScript 驾轻就熟，清楚整个网站开发流程。

刚毕业的那一两年，我从事网页设计工作，用AI和Photoshop把需求展现在效果图之上。

从2013年开始，我便专注于前端领域，经过多年的工作与学习，让我在 jQuery 、 Bootstrap 、 Node.js 、 D3.js 、 Snap.svg 、 PHP 等技术点都有涉略。我偏向于 git 工具控制项目版本，我习惯于开源免费的 Atom 现代编辑器。此外我每天都用 firebug 来调试网页，YSlow、ESlint、Fiddler这些好工具有时真帮了我大忙。下班回到家，我就在个人电脑玩转Ubuntu系统，对Linux常用命令、终端 VIM 不会感到陌生。

这几年前端发展很快，在最近的一年时间里，我着重学习和运用 React 框架，使用 gulp 构建自动化工作流。

我注重细节，拥有优雅的编程风格 ...

代码就是财富。代码是写给人看的，为方便其他同事阅读、维护，我在开发过程中，除了编写注释之外，通常牢记简单的两点：

  • 缩进 -- 两个空格
  • 命名 -- class使用减号（select-all-btn），js变量使用驼峰（eleSelectAllBtn）

... 以及敏捷的逻辑思维

some projects

  • 俄罗斯方块

    没有尝试自己写过一个游戏，算不上热爱编程。还记得小时候我们一班农村孩子排排坐，拿着小游戏机就是为这俄罗斯方块着迷。

    该项目是一个业余项目，主要是在canvas上绘图，还涉及一些PHP，MySQL知识，用来保存游戏分数，统计pv之类。

  • 九空格

    把1~-9填入九宫格内,使横竖斜三个数的和等于15. 通过最常见的枚举法得到答案。

  • 备忘录

    用React编写的单页app备忘录。使用localStorage存储数据，我在我的工作电脑上日日运行它。

  • swiper lite

    强大的swiper大家可能都听说过，但我写的是最简版的。仅支持手机，不依赖任何库，纯原生，代码小。此外启用硬件加速，很流畅。

  • 七巧

    不知为啥，我就对七巧板情有独钟。使用CSS3，你可以摆出很多有意思的图案。

  • xposed后台

    一个后台界面优化的案例，我接手时已有雏形，经过我的设计及重构优化，经理表示很喜欢。

  • rom下载站

    rom资源的下载站，同时包含有刷机教程。

  • chrome扩展：baiduCAl

    谷歌浏览器绝对是最有前途的桌面浏览器。百度则是国内访问量最高的网站之一。很遗憾，我没有发现一个键盘快捷键能将焦点定位到百度搜索框去，于是我写了一个chrome扩展。

  • Flexigrid贡献代码

    Flexigrid是一个表格插件。但是它有缺点--鼠标拖拽列时会选中表格的数据，最终以反色显示。我修改了作者插件源码并申请pull request.
